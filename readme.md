# JS Web scrapper for wp plugins changelogs

    App use [puppeteer](https://pptr.dev) and [expressjs](https://expressjs.com/)

## Start project
 
`npm install`

`node server`

## Structure 
    /
    | node_modules/
    | src/
    |     gravityform/
    |         index.js
    |     wp-rocket/
    |         index.js
    | server.js

## API routes:

- /wp-rocket
- /gravityform
