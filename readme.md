# JS Web scrapper for wp plugins changelogs

    App use puppeteer and expressjs

## Start project
 
`npm install`

`node server`

## Structure 
    /
    | node_modules/
    | src/
    |     gravityform/
    |         index.js
    |     wp-rocket/
    |         index.js
    | server.js

## API routes:

- /wp-rocket
- /gravityform
